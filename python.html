What is Python?

Python is a general-purpose, dynamically typed, high-level, interpreted programming language. It emphasizes code readability and allows programmers to express concepts with fewer lines of code compared to other languages.

Python was conceived in the late 1980s by Guido van Rossum. The first version, Python 0.9.0, was released in 1991. Since then, Python has gained immense popularity and is widely used in various domains due to its simplicity and versatility.

Python Features and Characteristics

  • Easy to Learn and Use: Python's syntax is straightforward and emphasizes readability, making it accessible for beginners.
  • Expressive Language: Python allows developers to write clear and logical code for small and large-scale projects.
  • Interpreted Language: Python code is executed line by line at runtime by the interpreter.
  • Dynamic Typing: Python uses dynamic typing, enabling developers to write code without declaring the type of variables.
  • Rich Standard Library: Python's extensive standard library supports various tasks such as string operations, file I/O, and web programming.
  • Support for Multiple Paradigms: Python supports procedural, object-oriented, and functional programming paradigms.
  • Platform Independent: Python code runs unchanged on various platforms, including Windows, macOS, and Linux.
  • Vibrant Community and Ecosystem: Python has a large community of developers contributing libraries, frameworks, and tools.

Python Data Types and Structures

Python provides various built-in data types and structures:

  • Strings: Sequence of characters enclosed in quotes, used for text manipulation.
  • Numbers: Integers, floating-point numbers, and complex numbers for mathematical operations.
  • Booleans: True or False values used in logical operations.
  • Lists: Ordered collection of items that are mutable and can contain mixed data types.
  • Tuples: Immutable ordered collection of items, similar to lists but cannot be modified after creation.
  • Sets: Unordered collection of unique items, useful for operations like membership testing and elimination of duplicates.
  • Dictionaries: Key-value pairs that are unordered, mutable, and indexed, used for storing data in a structured manner.
  • Arrays: Supported by libraries like NumPy, used for efficient numerical computations and data manipulation.

Python Lists, Tuples, and Sets

Python provides versatile data structures for storing collections of data:

  • Lists: Ordered and mutable collections of items, defined using square brackets []. Example: numbers = [1, 2, 3, 4, 5].
  • Tuples: Immutable collections of items, defined using parentheses (). Example: coordinates = (10, 20).
  • Sets: Unordered collections of unique items, defined using curly braces {}. Example: unique_numbers = {1, 2, 3, 4, 5}.

Python Dictionaries and Arrays

Other essential data structures in Python include dictionaries and arrays:

  • Dictionaries: Key-value pairs enclosed in curly braces {}, allowing fast lookups and mappings. Example: person = {'name': 'John', 'age': 30}.
  • Arrays: Numeric arrays supported by libraries like NumPy, offering efficient storage and operations for numerical data. Example: import numpy as np; arr = np.array([1, 2, 3]).

Python Type Casting

Type casting in Python allows conversion between different data types:

  • Implicit Type Conversion: Automatic conversion performed by Python based on the context. Example: x = 10 + 5.5.
  • Explicit Type Conversion: Manual conversion using built-in functions like int(), float(), and str(). Example: age = int("25").

Python Functions

Functions in Python are defined using the def keyword and can have parameters and return values:

def greet(name):
    return f"Hello, {name}!"

message = greet("Alice")
print(message)

This defines a simple function greet that takes a name as an argument and returns a greeting message.

Python Modules and Packages

Python supports modular programming through modules and packages:

  • Modules: Files containing Python code, which can be imported and used in other Python scripts. Example: import math; print(math.sqrt(16)).
  • Packages: Collections of modules organized in directories. Packages allow structuring a Python project into multiple modules. Example: import numpy as np; arr = np.array([1, 2, 3]).

Python Object-Oriented Programming

Python supports object-oriented programming (OOP) with classes and objects:

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hello, my name is {self.name} and I am {self.age} years old."

person = Person("Alice", 30)
print(person.greet())

This defines a class Person with attributes name and age, and a method greet to introduce the person.
