What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS was first proposed by Håkon Wium Lie in 1994 and has since become the standard styling language for web pages. It allows developers to control the layout, colors, fonts, and overall appearance of web pages.

CSS Selectors

CSS selectors are patterns used to select and style HTML elements:

  • Element Selector: Selects elements by tag name
  • Class Selector: Selects elements with a specific class
  • ID Selector: Selects elements with a specific ID
  • Attribute Selector: Selects elements with specific attributes
  • Pseudo-class Selector: Selects elements in specific states
/* Element selector */
p {
    color: blue;
}

/* Class selector */
.highlight {
    background-color: yellow;
}

/* ID selector */
#header {
    font-size: 24px;
}

CSS Box Model

The CSS box model consists of:

  • Content: The actual content of the element
  • Padding: Clear area around the content
  • Border: A border around the padding
  • Margin: Clear area outside the border
.box {
    width: 200px;
    height: 100px;
    padding: 20px;
    border: 2px solid black;
    margin: 10px;
    background-color: lightblue;
}

CSS Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns:

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    flex-direction: row;
}

.item {
    flex: 1;
    margin: 10px;
}

Flexbox makes it easy to create responsive layouts and center content both horizontally and vertically.

CSS Grid

CSS Grid is a two-dimensional layout system for creating complex web layouts:

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: auto;
    gap: 20px;
}

.grid-item {
    background-color: #f0f0f0;
    padding: 20px;
    text-align: center;
}

Grid allows for precise control over both rows and columns, making it perfect for complex layouts.
