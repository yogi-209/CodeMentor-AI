What is C++?

C++ is a general-purpose programming language widely used for competitive programming. It features imperative, object-oriented, and generic programming capabilities. C++ runs on multiple platforms like Windows, Linux, Unix, and Mac. It is efficient and powerful, commonly used in GUI platforms, 3D graphics, and real-time simulations.

Here's a simple "Hello World" program in C++: 

#include<iostream>
using namespace std;
int main()
{
    cout << "Hello World";
    return 0;
}
C++ is an Object-Oriented Programming Language.

Objects and Classes

Object-oriented programming uses objects in programming, aiming to implement real-world entities like inheritance, hiding, and polymorphism. The main goal is to bind data and functions together, ensuring that no other part of the code can access this data except for the associated function.

In C++, classes act as blueprints for creating objects. An object is an instance of a class, encapsulating data and methods (functions) that operate on the data. For example: 

class Car {
public:
    string brand;
    string model;
    int year;
    
    void displayInfo() {
        cout << "Car: " << brand << " " << model << " (" << year << ")" << endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.model = "Corolla";
    myCar.year = 2022;
    myCar.displayInfo();
    return 0;
}
Here, Car is a class with attributes brand, model, and year, and a method displayInfo() to display car details.

Inheritance

Inheritance is a fundamental feature of OOP where a class (subclass or derived class) can inherit properties and behavior (methods) from another class (superclass or base class). This promotes code reusability and supports the "is-a" relationship. Example: 

class Animal {
public:
    void makeSound() {
        cout << "Animal makes a sound" << endl;
    }
};

class Dog : public Animal {
public:
    void makeSound() {
        cout << "Dog barks" << endl;
    }
};

int main() {
    Dog myDog;
    myDog.makeSound(); // Output: Dog barks
    return 0;
}

Here, Dog inherits from Animal, and makeSound() is overridden to specify a dog's bark.

Polymorphism

Polymorphism allows objects of different classes to be treated as objects of a common parent class. This promotes flexibility and reusability in code. There are two types of polymorphism in C++: compile-time (function overloading) and runtime (function overriding). 

class Shape {
public:
    virtual void draw() {
        cout << "Drawing a shape" << endl;
    }
};

class Circle : public Shape {
public:
    void draw() override {
        cout << "Drawing a circle" << endl;
    }
};

int main() {
    Shape* shape = new Circle();
    shape->draw(); // Output: Drawing a circle
    delete shape;
    return 0;
}

Here, Shape is a base class with a virtual method draw(), and Circle is a derived class overriding draw() to draw a circle.

Abstraction

Abstraction focuses on hiding complex implementation details and showing only the necessary features of an object. It allows programmers to work with high-level ideas rather than low-level details. 

class Animal {
public:
    virtual void makeSound() = 0; // Pure virtual function
};

class Dog : public Animal {
public:
    void makeSound() override {
        cout << "Dog barks" << endl;
    }
};

int main() {
    Animal* animal = new Dog();
    animal->makeSound(); // Output: Dog barks
    delete animal;
    return 0;
}

Here, Animal is an abstract class with a pure virtual function makeSound(). Dog inherits from Animal and implements makeSound().

Encapsulation

Encapsulation bundles data (attributes) and methods (functions) into a single unit called a class. It protects data from unauthorized access and modification by providing public interfaces (methods) for interaction. 

class Employee {
private:
    string name;
    int age;
    double salary;

public:
    // Constructor
    Employee(string n, int a, double s) : name(n), age(a), salary(s) {}

    // Getter methods
    string getName() {
        return name;
    }

    int getAge() {
        return age;
    }

    double getSalary() {
        return salary;
    }

    // Setter methods
    void setName(string n) {
        name = n;
    }

    void setAge(int a) {
        age = a;
    }

    void setSalary(double s) {
        salary = s;
    }

    void displayInfo() {
        cout << "Employee: " << name << ", Age: " << age << ", Salary: " << salary << endl;
    }
};

int main() {
    Employee emp("John Doe", 30, 50000);
    emp.displayInfo();
    emp.setAge(31);
    emp.setSalary(55000);
    emp.displayInfo();
    return 0;
}

Here, Employee is a class with private attributes name, age, and salary. Public getter and setter methods provide controlled access to these attributes.
